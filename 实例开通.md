# 阿里云轻量/ECS开通与配置指南

本项目现在全面支持 **阿里云国内站（人民币结算）** 及 **阿里云国际站（美元结算）** 的实例监控。
无论您是利用国际版的 CDT 免费额度，还是国内版的按量计费套餐，皆可通过本监控脚本保障流量安全，防流超标扣费。

## 场景 A: 阿里云国际站 (美金结算) - 零成本畅享 200G/月 BGP 路线
如果您是国际站用户，您可以利用 **CDT（云数据传输）** 的 200GB 免费发往非中国内地的流量额度，配合 **抢占式实例（Preemptible Instance）** 获得低成本的高质量网络。

### 1. 开启 CDT 共享流量包 (至关重要)
如果不开启此项，流量将按昂贵的标准价格计费！
1. 登录阿里云国际站控制台 [alibabacloud.com](https://www.alibabacloud.com/)
2. 在顶部搜索框输入 **“CDT”**，点击进入 **“云数据传输”** 概览页。
3. 检查页面上的“公网流量”或“云数据传输”子项，点击右上角的 **“一键升级”** 或 **“开通服务”** 按钮完成升级。

### 2. 极简镜像准备 (推荐 Alpine)
为了进一步降低磁盘费用和系统资源占用，推荐使用超轻量级的 **Alpine Linux**。
- 您可以前往本项目 Releases 页面下载特制的 ISO 镜像 `alpine-virt-3.23.2-x86_64.iso`。
- 在与准备购买 ECS 相同的地域开通 OSS，将该 `.iso` 镜像上传并导入为 ECS 自定义镜像。

### 3. 创建“抢占式”实例及 EIP
1. 进入 ECS 购买页面，选择 **抢占式实例 (Pay-As-You-Go)**。
2. 地域选择 **中国香港** 等海外节点，规格上寻找入门级（例如 `e` 系列，通常配置费用低至 $0.001/小时）。
3. 镜像选择刚刚导入的脱水版系统，磁盘大小拉到最低。
4. **不勾选**分配公网 IPv4 地址。
5. 实例创建完成后，单独前往 [弹性公网 IP (EIP) 控制台](https://vpc.console.alibabacloud.com/eip) 申请一个 EIP，加入免费的 CDT 计费模式并绑定到该实例即可。

> **⚠️ 注意：国际站 CDT 采用“出入取大值”计费，下载和上传中较大的一方将作为扣除的标准，并非完全的双向免费！**

## 场景 B: 阿里云国内站 (人民币结算) - 常规按量付费指引
对于阿里云国内站（中国站）用户，您可以正常开通按量付费或包年包月的轻量应用服务器与 ECS。由于国内版不享有相同的 CDT 免费套餐政策，流量将统一按照人民币单价实时结算，因此配置严格的监控尤为重要。

### 1. 创建按量付费实例
1. 使用账号登录 [阿里云中国站 (aliyun.com)](https://www.aliyun.com/)。
2. 导航至 **云服务器 ECS** -> **实例** -> **创建实例**。
3. 付费模式务必选择 **按量付费**。
4. 区域根据您的业务选择（如北京、上海、香港、新加坡等），脚本已完全兼容全球地域的支持。

### 2. 网络与计费配置
1. 分配公网 IPv4 地址，并**务必选择“按使用流量计费”**（监控脚本通过 API 抓取的数据也基于此维度运算），请勿选择固定带宽。
2. 带宽峰值可拉高（如 100Mbps）。
3. 核对下方的预估按量价格，确认购买。

## 准备对接监控监控参数
部署 `install.sh` 时，您需要准备以下字段：
- **备注名**：自定义以便在 Telegram 机器人中识别（如 HK-Server）。
- **AccessKey 和 Secret**：进入右上角头像 -> AccessKey 管理，创建并保存 AK 和 SK。为安全起见，建议授权 `AliyunECSReadOnlyAccess`、`AliyunBSSReadOnlyAccess`、`AliyunCDTReadOnlyAccess` 和 ECS 启动/停止权限。
- **账号类型（新增特性）**：安装过程中会询问，请根据您所属站别选择：
  - `1`：国内区 (`business.aliyuncs.com`，人民币 ￥ 结算)
  - `2`：国际区 (`business.ap-southeast-1.aliyuncs.com`，美元 $ 结算)
- **ECS 实例 ID**：在 ECS 列表页复制 `i-xxxxxxx` 的字符串。

准备就绪后，直接执行仓库根目录下的 `install.sh` 即可完成部署！
